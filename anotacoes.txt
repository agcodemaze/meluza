Executar os testes: bash -> ./bin/phpunit

$this->assertEquals($expected, $actual): Verifica se dois valores são iguais. Funciona bem para strings, números, arrays e objetos.
$this->assertSame($expected, $actual): Similar ao assertEquals, mas verifica se os valores e os tipos são idênticos. É mais estrito e útil para evitar bugs com tipos.
$this->assertTrue($condition): Verifica se a condição é true.
$this->assertFalse($condition): Verifica se a condição é false.
$this->assertNull($variable): Verifica se a variável é null.
$this->assertNotNull($variable): Verifica se a variável não é null.
$this->assertArrayHasKey($key, $array): Verifica se uma chave existe em um array.
$this->assertObjectHasAttribute($attribute, $object): Verifica se uma propriedade existe em um objeto.
$this->assertClassHasAttribute($attribute, $class): Verifica se uma propriedade estática ou de classe existe.
$this->assertFileExists($filename): Verifica se um arquivo existe.
$this->assertGreaterThan($expected, $actual): Verifica se o valor atual é maior que o esperado.
$this->assertLessThan($expected, $actual): Verifica se o valor atual é menor que o esperado.
$this->assertEmpty($variable): Verifica se uma variável está vazia (por exemplo, "", [], null).
$this->assertNotEmpty($variable): Verifica se uma variável não está vazia.
$this->assertIsArray($variable): Verifica se a variável é um array.
$this->assertIsString($variable): Verifica se a variável é uma string.
$this->assertIsInt($variable): Verifica se a variável é um inteiro.
